# Kalp Cotspin Denim Library

A professional React-based Denim Fabric Information Portal developed for Kalp Cotspin.

---

## Features

- Modern Material UI Dashboard
- Responsive Layout
- Denim Fabric Categories
- Fabric Specification Tables
- Search & Filters
- Image Gallery
- Fabric Detail Pages
- Professional Navigation
- Expandable Categories
- GitHub Ready
- Netlify Ready

---

## Tech Stack

- React 18
- Vite
- Material UI
- React Router
- Emotion
- JavaScript (ES6+)

---

## Installation

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Production Build

```bash
npm run build
```

Preview Production

```bash
npm run preview
```
